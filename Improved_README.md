# Library Management System (DBMS Project)

## Introduction

The Library Management System is a database-driven application designed
to manage books, members, publishers, librarians, and borrowing
operations efficiently. The project demonstrates the practical
implementation of Database Management System concepts using a relational
database model while maintaining data consistency, integrity, and
efficient retrieval of information.

## Objectives

-   Manage book inventory effectively
-   Track book issue and return transactions
-   Maintain member records
-   Store publisher and librarian information
-   Calculate fines and monitor borrowing history
-   Ensure relational integrity between all entities

## Technologies Used

-   MySQL
-   SQL
-   Database Management System (DBMS)

-   Many-to-Many Relationships
-   Normalization up to Third Normal Form (3NF)
-   Views

## Normalization

The database schema is normalized up to Third Normal Form (3NF) to
minimize data redundancy and improve consistency while preserving
referential integrity.

This project demonstrates the design and implementation of a Library
Management System using DBMS principles. It showcases relational schema
design, SQL operations, constraints, normalization, and query
optimization techniques in a structured and practical manner.
